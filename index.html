How to use

HTML:

<div class="hohoacc">
  <div class="panel">
    ... your content ...
  </div>
  <div class="panel">
    ... your content ...
  </div>
  <div class="panel">
    ... your content ...
  </div>
  <div class="panel">
    ... your content ...
  </div>
</div>

JavaScript:

$(".hohoacc").hoHoAcc();

This is the basic usage example.

You specify the container <div class="hohoacc"> and add panels <div class="panel"> with content inside of it.

In your javascript you simply invoke the plugin by $(".hohoacc").hoHoAcc();

On the right you can find the list of avaliable options you can pass to constructor.

All options also can be passed as data attributes in html (notice that options all lowercased here):

<div class="hohoacc" data-panelwidth="400" data-gapmax="10">
</div>

Also, a number of extra classes is avaliable to align the contents of panels. Check them on the right.

Check the examples on that page for more detailes.

There is a number of options that can be passed to constructor as object:

$(".hohoacc").hoHoAcc({
  panelWidth: 500,
  gapMax: 10,
  gapMin: 20,
  speed: 300,
  easing: "linear"
});
  • panelWidth specifies the width of the panel when you hover it. It can be a number or a string containing the relative width in percents (e.g. panelWidth: "80%"). Defaults to 70%.
  • gapMax specifies the space beetween the panels when nothing is hovered. It's a number. Defaults to 0.
  • gapMin specifies the space beetween the panels when one of them is hovered. Defaults to 0.
  • speed specifies animation duration.
  • easing specifies the easing function name to be applied to animation.

Additional classes avaliable for panels:

  • .panel-bg-centered forces the background image of panel to be aligned center (warn: using !importaint).
  • .panel-bg-righted forces the background image of panel to be aligned right (warn: using !importaint).
  • .panel-centered forces the contents of the panel to align center.
  • .panel-righted forces the contents of the panel to align right.

Two events are fired on the accordion container node (e.g. on $(".hohoacc")):

  • expandPanelStart - gets fired when a panel starts to expand.
  • collapsePanelStart - gets fired when a panel starts to collapse.

You can subscribe to those and do some extra work, like in the last example on this page:

$("#example4").on("expandPanelStart",
function(e) {
  $(this).find("h2").addClass("dull");
});

$("#example4").on("collapsePanelStart",
function(e) {
  $(this).find("h2").removeClass("dull");
});
